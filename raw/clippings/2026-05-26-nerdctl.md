# nerdctl 容器运行时

nerdctl 是一个 Docker 兼容的容器运行时 CLI 工具，由 containerd 社区维护。

## 简介

| 项目 | 说明 |
|------|------|
| **定位** | containerd 的命令行工具，兼容 Docker CLI |
| **特点** | 支持 Docker 大部分命令，同时提供额外高级功能 |
| **适用场景** | K8s 环境（containerd）、rootless 容器、沙箱隔离 |

## 与 Docker 的区别

### 架构对比

| 组件 | Docker | nerdctl |
|------|--------|---------|
| **CLI** | docker | nerdctl |
| **守护进程** | dockerd | 直接调用 containerd |
| **容器运行时** | containerd → runc | runc |
| **K8s 兼容性** | 需 dockershim 桥接 | 直接兼容 |

### 核心差异

| 特性 | Docker | nerdctl |
|------|--------|---------|
| **rootless 支持** | 受限（需额外配置） | 原生支持 |
| **镜像签名** | 不支持 | nerdctl sign 支持 |
| **加密镜像** | 不支持 | nerdctl image encrypt/decrypt |
| **构建缓存去重** | 不支持 | nerdctl build --duplicate-ref |
| **Gfarm 去重** | 不支持 | 支持镜像层全局去重 |
| **沙箱隔离** | 受限 | 原生 namespace isolation |
| **内存占用** | 较高 | 较低 |

| 2022 年 | v1.0 | 生产可用，完整 Docker 兼容 |

## 安装方式

```bash
# Linux/macOS (二进制安装)
curl -sSL https://raw.githubusercontent.com/containerd/nerdctl/main/install.sh | sh

# macOS (Homebrew)
brew install nerdctl

# 从 GitHub Releases 下载
# https://github.com/containerd/nerdctl/releases
```

## 命令对照

```bash
# Docker                    # nerdctl
docker run                 # nerdctl run
docker build -t img .      # nerdctl build -t img .
docker ps                  # nerdctl ps
docker compose up -d        # nerdctl compose up -d
docker pull nginx          # nerdctl pull nginx
                             # nerdctl encrypt image     # 镜像加密
                             # nerdctl sign image       # 镜像签名
```
