# 营运能力指标

## 定义

营运能力指标衡量企业使用资产的效率，包括存货周转、应收账款回收、资产利用率等。

## 一、存货周转率

### 计算公式

```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 存货周转率
```

### 判断标准

| 周转天数 | 含义 |
|----------|------|
| < 30天 | 优秀，产品畅销 |
| 30-90天 | 正常 |
| 90-180天 | 偏高，可能滞销 |
| > 180天 | ⚠️ 严重滞销，减值风险 |

### 警示信号

- 周转天数突然拉长
- 存货增速 > 营收增速（积压）
- 存货与营收反方向变化

---

## 二、应收账款周转率

### 计算公式

```
应收账款周转率 = 营收 / 平均应收账款
应收账款周转天数 = 365 / 应收账款周转率
```

### 判断标准

| 周转天数 | 含义 |
|----------|------|
| < 30天 | 优秀，回款快，对下游话语权强 |
| 30-90天 | 正常 |
| 90-180天 | 偏高，回款慢 |
| > 180天 | ⚠️ 坏账风险 |

### 关键警示

```
应收账款增速 > 营收增速 = 放宽信用
```

这意味着公司在"卖货"但"收不到钱"，利润质量下降。

---

## 三、总资产周转率

### 计算公式

```
总资产周转率 = 营收 / 平均总资产
```

### 判断标准

| 周转率 | 含义 |
|--------|------|
| > 1 | 优秀，资产利用效率高 |
| 0.5-1 | 正常 |
| < 0.5 | 低效，资产闲置或利用率低 |

### 注意事项

- **同行对比**：不同行业资产轻重不同，必须行业内比较
- **趋势更重要**：周转率持续提升 = 效率改善

---

## 四、运营能力综合评估

| 指标 | 优秀 | 正常 | 警示 |
|------|------|------|------|
| 存货周转天数 | < 30天 | 30-90天 | > 90天 |
| 应收账款天数 | < 30天 | 30-90天 | > 90天 |
| 总资产周转率 | > 1 | 0.5-1 | < 0.5 |

## 东方电气运营能力

| 指标 | 说明 | 风险点 |
|------|------|--------|
| 存货 | 216.85亿（占净资产 55.9%）| ⚠️ 占比较高，周转偏慢 |
| 应收账款 | 125.45亿 | ⚠️ 占流动资产比重大 |

**结论**：东方电气属于重资产、长周期行业，存货和应收账款规模大是行业特性，但需关注周转效率。
