# 成长性指标

## 定义

成长性指标衡量企业营收、利润等核心指标的增速，是评估企业"长大"能力的工具。

## 一、营收增速

### 计算公式

```
营收增速 = (本期营收 - 上期营收) / 上期营收 × 100%
```

### 判断标准

| 营收增速 | 含义 |
|----------|------|
| > 30% | 高增长 |
| 10-30% | 稳健增长 |
| 5-10% | 缓慢增长 |
| < 5% | 停滞/衰退 |

---

## 二、净利润增速

### 计算公式

```
净利润增速 = (本期净利润 - 上期净利润) / 上期净利润 × 100%
```

### 营收增速 vs 净利润增速

| 关系 | 含义 |
|------|------|
| 净利润增速 > 营收增速 | 规模效应，控费能力强 |
| 净利润增速 = 营收增速 | 稳定增长 |
| 净利润增速 < 营收增速 | 扩张让利 or 成本上升 |

---

## 三、3年复合增速 (CAGR)

### 计算公式

```
CAGR = (期末值 / 期初值)^(1/年数) - 1
```

### 为什么用 CAGR

1. **平滑周期波动**：单年增速可能受基数影响
2. **更真实**：反映持续增长能力
3. **跨行业可比**：标准化后的增速

### 示例

```
某公司：
- 2022年净利润：10亿
- 2025年净利润：17.2亿
- CAGR = (17.2/10)^(1/3) - 1 = 20%

结论：3年复合增速20%，持续增长能力强
```

---

## 四、成长性指标判断框架

| 维度 | 优秀 | 正常 | 警示 |
|------|------|------|------|
| 营收增速 | > 20% | 10-20% | < 10% |
| 净利润增速 | > 20% | 10-20% | < 10% |
| 3年CAGR | > 20% | 10-20% | < 10% |
| 增速稳定性 | 持续 | 波动 | 下滑 |

## 五、成长性陷阱

### 1. 低基数陷阱

```
去年利润1亿，今年2亿，增速100%
但实际规模很小，不值得关注
```

### 2. 并购虚增陷阱

```
通过收购快速增厚利润和营收
但商誉暴雷风险大
信号：商誉/净资产 > 30%
```

### 3. 应收驱动陷阱

```
营收和利润都增长
但应收账款增速更快
结果是"纸面富贵"
```
