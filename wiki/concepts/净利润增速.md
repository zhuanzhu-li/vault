# 净利润增速

## 定义

净利润增速是衡量公司盈利能力增长速度的指标，反映公司业务扩张和盈利改善的能力。

## 计算公式

```
净利润增速 = (本期净利润 - 上期净利润) / 上期净利润 × 100%
```

**复合年增长率（CAGR）：**
```
CAGR = (终值 / 初值)^(1/年数) - 1
```

## 增速类型

| 类型 | 计算方式 | 特点 |
|------|---------|------|
| **同比增速** | 与去年同期比 | 季节性影响小 |
| **环比增速** | 与上季度比 | 反映短期变化 |
| **CAGR** | 复合年均增长 | 长期趋势更可靠 |

## 判断标准

| 增速范围 | 评价 | 说明 |
|----------|------|------|
| < 0% | 负增长 | 盈利能力下降 |
| 0-10% | 低速增长 | 成熟行业常见 |
| 10-20% | 稳健增长 | 较好的成长性 |
| 20-50% | 高速增长 | 成长股特征 |
| > 50% | 爆发式增长 | 需警惕可持续性 |

## 关键注意事项

1. **质量重于速度**：关注净利润含金量（现金流匹配度）
2. **可持续性**：一次性收益不代表长期增长
3. **行业对比**：不同行业增速标准不同
4. **基数效应**：低基数容易产生高增速

## 与估值的关系

- **高增速支撑高PE**：市场愿意为增长买单
- **增速下滑需警惕**：PE可能随之下降
- **PEG结合**：用PEG判断增速与估值是否匹配
