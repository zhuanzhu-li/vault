# 成长股投资

## 定义

成长股投资是一种投资策略，专注于投资具有高增长潜力、未来盈利预期持续提升的公司。

## 成长股特征

| 特征 | 说明 |
|------|------|
| **高净利润增速** | 通常20%以上 |
| **高PE估值** | 市场给予溢价 |
| **轻资产** | 无形资产占比高 |
| **行业领先** | 具有竞争优势 |
| **市场空间大** | 赛道有增长潜力 |

## 投资步骤

### 1. 筛选阶段
- 连续3年净利润增速 > 20%
- ROE > 15%
- 毛利率稳定或提升

### 2. 估值阶段
- 使用PEG判断是否合理
- 对比行业平均PE
- 考虑未来3年增长预期

### 3. 跟踪阶段
- 季度财报验证增长
- 行业竞争格局变化
- 管理层战略调整

## PEG应用场景

| PEG区间 | 行动建议 |
|---------|---------|
| < 0.5 | 强烈关注，寻找买点 |
| 0.5-1 | 合理估值，逢低买入 |
| 1-1.5 | 谨慎，等待更好价格 |
| > 2 | 风险较高，观望 |

## 风险点

1. **增速不及预期**：高PE下增速下滑会导致股价大跌
2. **估值泡沫**：过高预期可能破裂
3. **竞争加剧**：护城河可能被突破
4. **宏观环境**：经济下行影响成长

## 常见误区

- 只看增速不看估值
- 忽视可持续性
- 追涨杀跌
