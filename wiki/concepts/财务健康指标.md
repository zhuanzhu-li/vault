# 财务健康指标

## 定义

财务健康指标用于评估企业的偿债能力、资金周转能力和财务稳健性，是识别企业风险的重要工具。

## 一、资产负债率

### 计算公式

```
资产负债率 = 总负债 / 总资产 × 100%
```

### 判断标准

| 行业 | 健康范围 | 说明 |
|------|---------|------|
| 金融（银行/保险）| 80-90%+ | 行业特性，高杠杆 |
| 房地产 | 70-80% | 资金密集型 |
| 制造业 | < 60% | 稳健水平 |
| 互联网/轻资产 | < 40% | 轻资产模式 |

### 注意事项

1. **区分有息负债 vs 无息负债**：无息负债（应付账款）风险较低
2. **负债结构**：短期负债占比高 = 偿债压力大
3. **趋势比绝对值重要**：资产负债率持续上升 = 风险积累

---

## 二、流动比率

### 计算公式

```
流动比率 = 流动资产 / 流动负债
```

### 判断标准

| 流动比率 | 含义 |
|----------|------|
| > 2 | 财务稳健 |
| 1.5-2 | 正常 |
| 1-1.5 | 勉强 |
| < 1 | ⚠️ 流动性危机信号 |

### 含义

流动比率 > 1 表示流动资产能覆盖流动负债；< 1 表示短期内可能还不起债。

---

## 三、速动比率

### 计算公式

```
速动比率 = (流动资产 - 存货) / 流动负债
```

### 判断标准

| 速动比率 | 含义 |
|----------|------|
| > 1 | 短期偿债能力强 |
| < 1 | 存货变现慢，存在流动性压力 |

### 为什么剔除存货

存货变现需要时间，且可能贬值，因此速动比率比流动比率更严格地衡量短期偿债能力。

---

## 四、现金流覆盖倍数

### 计算公式

```
现金流覆盖倍数 = 经营现金流(CFO) / 有息负债
```

### 判断标准

| 倍数 | 含义 |
|------|------|
| > 1 | 一年内可还清债务 |
| 0.5-1 | 覆盖能力一般 |
| < 0.5 | ⚠️ 债务压力较大 |

### 关键指标

```
CFO > 净利润 = 利润质量高（真赚钱）
CFO < 净利润 = 利润质量低（可能是纸面富贵）
```

---

## 五、财务健康检查清单

```
□ 资产负债率是否在行业合理范围内？
□ 流动比率是否 > 1.5？
□ 速动比率是否 > 1？
□ CFO 是否 > 净利润？
□ 有息负债是否持续增长？
□ 短期借款占比是否过高？
```

## 六、东方电气财务健康

| 指标 | 数据 | 评价 |
|------|------|------|
| 资产负债率 | 69.62% | ⚠️ 偏高，需关注 |
| 流动比率 | 1.05 | 勉强 |
| 速动比率 | 0.81 | ⚠️ 偏低 |
| CFO/净利润 | > 1 | ✅ 优秀 |
| 货币资金 | 268.56亿 | ✅ 储备充足 |

**结论**：现金流优秀，但负债率和存货周转需关注。
