# 职业回报计算公式

## 公式

```
回报 = (钱 + 成长 + 舒适度) / 投入时间
```

## 要素说明

- **钱**：薪资，市场通过钱告诉你值多少
- **成长**：技能提升、视野开阔、人脉积累
- **舒适度**：通勤、团队、业务、氛围
- **投入时间**：实际工作时长

## 关键观点

- 薪资涨幅低于30%时要谨慎评估是否有新价值
- 看似薪资持平但每天多睡1小时其实是涨薪
- 薪资涨了但每天多工作3小时其实是降薪
- 不要只看月薪数字，要综合计算
