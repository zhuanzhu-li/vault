# ROE (净资产收益率)

## 定义

净资产收益率（Return on Equity）是衡量公司用自有资本赚钱能力的核心指标，反映股东投入的资本能带来多少回报。

## 计算公式

```
ROE = 净利润 / 净资产（平均） × 100%
```

## 杜邦分析（ROE三分解）

ROE 可以分解为三个维度的乘积：

```
ROE = 净利率 × 资产周转率 × 权益乘数
      ↑       ↑        ↑
    赚钱能力  运营效率  杠杆水平
```

| 分解维度 | 说明 | 优 |
|---------|------|-----|
| **净利率** | 每元收入能赚多少净利润 | 越高说明定价权强 |
| **资产周转率** | 每元资产能产生多少收入 | 越高说明运营效率高 |
| **权益乘数** | 杠杆倍数 | 越高说明负债越多 |

## 判断标准

| ROE | 评价 | 说明 |
|-----|------|------|
| > 20% | 优秀 | 有护城河，盈利能力强 |
| 15-20% | 良好 | 稳健，可持续 |
| 10-15% | 一般 | 普通，需结合行业判断 |
| < 10% | 较差 | 资本使用效率低 |

## 关键注意事项

1. **高ROE + 高负债 = 低质量**：有些公司靠借债推高ROE，要警惕
2. **持续性 > 单年数**：连续多年 > 15% = 护城河信号
3. **同行对比**：不同行业ROE差异大，必须行业内比较
4. **ROE 趋势**：比绝对值更重要，持续上升趋势更值得关注

## 应用场景

- 筛选优质公司：ROE 持续 > 15%
- 识别陷阱：高ROE但利润下滑
- 杜邦分析：找出ROE来源（是盈利好、运营好还是杠杆高）
