# PE (市盈率)

## 定义

市盈率（Price-to-Earnings Ratio）是股价与每股收益的比值，衡量"按当前价格买入，多久能回本"。

## 计算公式

```
PE = 股价 / EPS
```

## 三种类型

| 类型 | 计算方式 | 用途 |
|------|---------|------|
| **静态PE** | 股价 / 上年EPS | 参考历史定价 |
| **动态PE** | 股价 / 今年预期EPS | 预期定价 |
| **TTM PE** | 股价 / 近4季度EPS | 当前定价 |

## 判断标准

| PE范围 | 适用场景 | 说明 |
|--------|---------|------|
| < 10 | 价值股 / 周期股低谷 | 可能低估，也可能是陷阱 |
| 10-20 | 合理偏低 | 性价比较好 |
| 20-30 | 合理 | 市场正常定价 |
| 30-50 | 成长股 | 高增长预期 |
| > 50 | 热点炒作 | 风险较大 |

## 行业差异

| 行业 | 合理PE | 原因 |
|------|--------|------|
| 银行 | 5-8 | 高杠杆，资产重 |
| 煤炭 | 5-10 | 周期性强，盈利不稳定 |
| 白酒 | 20-30 | 稳定高 ROE |
| 制造业 | 15-25 | 中等增长 |
| 科技/AI | 30-50 | 高增长预期 |

## 关键注意事项

1. **低PE可能是陷阱**：利润下滑中的低PE不是低估
2. **高PE可能是合理**：高增长预期可以支撑高PE
3. **必须结合增速**：PE要与净利润增速一起看
4. **同行对比**：同行业内比较更准确

## 与 ROE 的配合使用

| 组合 | 含义 |
|------|------|
| 低PE + 高ROE | 潜在价值股 |
| 高PE + 高ROE | 成长股，合理 |
| 高PE + 低ROE | 估值偏高 |
| 低PE + 低ROE | 价值陷阱 or 周期底部 |
