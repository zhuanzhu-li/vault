# 缓存雪崩

大量热点 key 在同一时间失效，导致数据库压力骤增。

## 原因

- 批量 key 设置相同过期时间
- Redis 服务宕机

## 解决方案

1. **随机过期时间**：在固定过期时间上加随机值
2. **多级缓存**：本地缓存 + 分布式缓存
3. **缓存预热**：提前加载热点数据
4. **熔断降级**：保护数据库
