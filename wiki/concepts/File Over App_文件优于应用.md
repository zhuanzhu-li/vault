# 文件优于应用

[[Steph Ango_斯蒂芬·安戈|斯蒂芬·安戈]]提出的哲学：持久的数字作品应存在于**你可控制的文件**中，而非终将消失的应用内部。

## 核心思想

- 优先将成果存为可移植的文件，而非锁在单一应用数据库或云端的数据。
- 选择能最大化阅读、迁移与保存自由的工具。
- 工具制造者应承认[[Software Ephemerality_软件易逝性|软件易逝性]]，并为[[User Data Ownership_用户数据所有权|用户数据所有权]]而设计。

## 为何重要

凿子不会比象形文字更长久。同理，创作应用不如几十年后仍能打开的[[Open File Formats_开放文件格式|开放文件格式]]中所编码的思想重要。参见[[Digital Durability_数字耐久性|数字耐久性]]。
