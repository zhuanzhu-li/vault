# 现金流分析

## 定义

现金流分析通过比较经营现金流（CFO）与净利润的关系，验证企业利润的"含金量"，识别"纸面富贵"。

## 三种现金流

| 类型 | 说明 | 含义 |
|------|------|------|
| **经营现金流 (CFO)** | 主业赚的钱 | 核心能力 |
| **投资现金流 (CFI)** | 资本开支/投资收益 | 扩张/收缩 |
| **融资现金流 (CFF)** | 借债/分红/融资 | 资金来源 |

## 核心指标

### CFO / 净利润

```
CFO / 净利润 > 1 = 利润质量高（真赚钱）
CFO / 净利润 < 1 = 利润质量低（可能是纸面富贵）
```

**判断标准**：
| 比值 | 含义 |
|------|------|
| > 1.2 | 优秀 |
| 0.8-1.2 | 正常 |
| < 0.8 | ⚠️ 利润质量存疑 |
| < 0.5 | ⚠️ 严重警示 |

### 自由现金流 (FCF)

```
FCF = CFO - 资本开支
```

**判断标准**：
| FCF | 含义 |
|-----|------|
| > 0 | 有结余，可用于分红/还债/扩张 |
| = 0 | 刚好平衡 |
| < 0 | 消耗现金，需要融资 |

## 利润质量陷阱

### 1. 应收账款陷阱

```
利润增加 + CFO下降 = 放宽信用、放量赊账
信号：应收账款增速 > 营收增速
```

### 2. 存货陷阱

```
利润增加 + CFO下降 = 存货积压
信号：存货增速 > 营收增速
```

### 3. 借债分红陷阱

```
CFO < 分红总额 = 借钱分红
结果：短期好看，长期暴雷
```

## 现金流健康检查清单

```
□ CFO > 净利润（利润质量）
□ CFO > 0（能自给自足）
□ CFO 持续增长（成长性好）
□ FCF > 0（自由现金流为正）
□ CFO 增速与净利润增速匹配（同步性）
```

## 东方电气现金流

| 指标 | 数据 | 评价 |
|------|------|------|
| CFO (2024) | 100.59亿 | ✅ 优秀 |
| 净利润 (2024) | 29.22亿 | - |
| CFO/净利润 | > 3 | ✅ 利润质量极高 |

**结论**：东方电气主业造血能力强，是优质特征之一。
