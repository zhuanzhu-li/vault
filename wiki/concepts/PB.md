# PB (市净率)

## 定义

市净率（Price-to-Book Ratio）是股价与每股净资产的比值，衡量市场对公司账面价值的评估。

## 计算公式

```
PB = 股价 / 每股净资产
```

## 判断标准

| PB | 含义 | 适用场景 |
|----|------|---------|
| < 1 | 破净 | 理论便宜，需验证资产质量 |
| 1-3 | 合理区间 | 大多数情况 |
| 3-5 | 溢价 | 品牌/技术护城河 |
| > 5 | 高溢价 | 高增长 or 强品牌 |

## 适用行业

### 适合用 PB 的行业

| 行业 | 原因 |
|------|------|
| **银行** | 资产是核心，账面价值稳定 |
| **保险** | 投资资产透明 |
| **房地产** | 土地/房产按成本计量 |
| **周期股** | PE失真时用PB |

### 不适合用 PB 的行业

| 行业 | 原因 |
|------|------|
| 互联网 | 轻资产，用户/数据不计入账面 |
| 软件 | 研发投入费用化，资产轻 |
| 服务 | 人力密集，资产轻 |

## PB 的局限性

1. **账面价值 ≠ 真实价值**：资产可能被高估或低估
2. **不反映未来盈利**：只看历史账面价值
3. **行业差异大**：不同行业PB不可比

## PB 与 ROE 的配合

| PB | ROE | 含义 |
|----|-----|------|
| 低 | 高 | 便宜 + 优质 = 潜在机会 |
| 高 | 高 | 贵但值得（护城河强）|
| 低 | 低 | 便宜但有问题 |
| 高 | 低 | 贵且差 = 卖出信号 |
