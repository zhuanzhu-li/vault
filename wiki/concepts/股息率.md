# 股息率

## 定义

股息率是股息与股价的比值，衡量持有股票获得的分红收益。

## 计算公式

```
股息率 = (每股分红 / 股价) × 100%
       = (分红总额 / 股本) / 股价
```

## 判断标准

| 股息率 | 含义 | 注意事项 |
|--------|------|---------|
| > 5% | 高股息 | 需验证持续性 |
| 3-5% | 合理偏高 | 稳健型偏好 |
| 1-3% | 正常 | 大多数股票 |
| < 1% | 低股息 | 成长型公司 |

## 股息率陷阱

### 1. 周期股陷阱

```
煤炭股高股息率往往是景气高点：
- 煤价高 → 利润高 → 分红高 → 股息率高
- 但股价可能已经跌了
- 等煤价下跌 → 利润暴跌 → 股息归零 → 股价暴跌
```

### 2. 借钱分红陷阱

```
公司借钱分红 → 股息率虚高 → 不可持续
信号：分红 > 经营现金流
```

### 3. 分光吃尽陷阱

```
分红率 > 100% → 入不敷出 → 消耗储备
```

## 优质高股息的特征

| 特征 | 说明 |
|------|------|
| 利润稳定 | 经营现金流 > 净利润 |
| 分红率合理 | 分红率 30-60% |
| 持续分红 | 连续多年稳定分红 |
| 行业稳定 | 非强周期行业 |

## 分红率

```
分红率 = 分红总额 / 净利润 × 100%
```

| 分红率 | 含义 |
|--------|------|
| < 30% | 留有余地，可持续 |
| 30-60% | 合理 |
| 60-80% | 偏高，注意持续性 |
| > 80% | 危险，可能不可持续 |

## 辩证看待股息

> 股息是试金石，不是目的地。
> 好的高股息 = 稳定现金流 + 合理分红率 + 持续增长
> 差的高股息 = 低估值陷阱 + 周期高点 + 虚假繁荣
