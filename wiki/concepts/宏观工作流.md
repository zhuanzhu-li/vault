# 宏观工作流

## 定义

**宏观框架是地图，宏观工作流是导航系统。**

宏观工作流是一套每日固定执行的分析流程，通过**固定的判断路径**实现稳定的交易决策，避免"随机判断"。区别于宏观框架（静态的认知结构），工作流是动态的、周期性的执行系统。

## 核心价值

| 没有工作流 | 有工作流 |
|-----------|---------|
| 每天问不同问题 | 每天问相同问题 |
| 结论随机波动 | 变化变得可见 |
| 每次重新开始 | 持续积累判断 |
| 追逐行情 | 理解底层逻辑 |

## 三步AI工作流（每日10分钟）

### Step 1 宏观变量更新

```
更新以下变量的最新趋势：
- 实际利率
- 美元指数
- 流动性环境
- 风险偏好

请指出最近变化最大的变量。
```

### Step 2 情景变化检测

```
当前市场是否出现变量背离？
例如：
- 利率上行但股市上涨
- 美元下跌但黄金下跌

如果存在，可能原因是什么？
```

### Step 3 风险检测

```
当前宏观逻辑最大的风险点是什么？
哪些数据最可能推翻当前结构？
```

## 完整四步工作流

| Step | 内容 | 核心问题 |
|------|------|---------|
| 1 | 宏观环境判断 | 利率/美元/流动性趋势 |
| 2 | 风险偏好判断 | 股市/信用/波动率结构 |
| 3 | 资产映射 | 黄金/美股/美债/商品 |
| 4 | 风险边界测试 | 什么情况下判断失效？ |

## 与[宏观框架](宏观框架)的关系

- [宏观框架](宏观框架) → 静态的认知地图（有哪些变量、变量间关系）
- **宏观工作流** → 动态的导航系统（每天如何更新判断）

工作流使框架活起来，成为可执行的系统。

## 关键原则

1. **固定路径**：每天用相同的流程
2. **对比基准**：今天的判断 vs 昨天的判断
3. **自我纠偏**：判断错三次可以，但第四次会修正
