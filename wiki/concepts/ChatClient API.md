# ChatClient API

[ChatClient API](ChatClient%20API.md) 是 [Spring AI](Spring%20AI.md) 中的一项功能，提供了一个流畅的、构建器风格的接口，用于与 AI 聊天模型进行通信。

它的设计在习惯用法上类似于现有的 [Spring Framework](Spring%20Framework.md) API（如 `WebClient` 和 `RestClient`），使 Java 开发人员能够直观地采用它。它抽象了与底层提供商（如 [OpenAI](OpenAI.md)）通信的复杂性。
