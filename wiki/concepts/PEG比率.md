# PEG比率

## 定义

PEG比率是市盈率（PE）与净利润增速（G）的比值，用于判断成长股是否被高估或低估。

## 计算公式

```
PEG = PE / (净利润增速 × 100)
```

**简化版**：
```
PEG = PE / 预期净利润增速（%）
```

## 判断标准

| PEG | 含义 | 行动建议 |
|-----|------|---------|
| < 0.5 | 严重低估 | 强烈买入信号 |
| 0.5-1 | 相对低估 | 买入 |
| = 1 | 估值合理 | 持有 |
| 1-2 | 相对高估 | 谨慎 |
| > 2 | 严重高估 | 卖出 |

## 实战应用

### 示例

```
股票A：
- PE = 30
- 预期净利润增速 = 40%
- PEG = 30 / 40 = 0.75

结论：PEG < 1，相对低估，值得买入
```

```
股票B：
- PE = 30
- 预期净利润增速 = 10%
- PEG = 30 / 10 = 3

结论：PEG > 2，严重高估，应该卖出
```

## 关键注意事项

1. **适用于成长股**：不适用于周期股、低速增长股
2. **增速要合理**：用预期增速，不是历史增速
3. **时间维度**：3年CAGR比单年增速更可靠
4. **行业差异**：高速增长行业PE高是正常的

## PEG 的局限性

1. **增速是预期**：存在不确定性
2. **不适用于周期股**：周期股PE低反而是卖点
3. **不适用于价值股**：低增速不等于没价值

## 与 PE 的区别

| 指标 | 看什么 | 适用场景 |
|------|--------|---------|
| PE | 静态贵贱 | 所有股票 |
| PEG | 相对贵贱 | 成长股 |
