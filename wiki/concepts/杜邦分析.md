# 杜邦分析

## 定义

杜邦分析（DuPont Analysis）是将 ROE 分解为三个核心指标的框架，用于理解企业盈利能力的真实来源，识别高 ROE 是来自"真本事"还是"高杠杆"。

## 核心公式

```
ROE = 净利率 × 资产周转率 × 权益乘数
```

## 三分解详解

### 1. 净利率（盈利能力）

```
净利率 = 净利润 / 营收
```

**含义**：每1元收入能赚多少净利润

**驱动因素**：
- 毛利率（定价能力）
- 费用率（成本控制）
- 税率（税收优惠）

**高质量信号**：净利率高且稳定

---

### 2. 资产周转率（运营效率）

```
资产周转率 = 营收 / 平均总资产
```

**含义**：每1元资产能产生多少收入

**驱动因素**：
- 存货周转速度
- 应收账款回收速度
- 固定资产利用率

**高质量信号**：周转率高且稳定 or 持续提升

---

### 3. 权益乘数（杠杆水平）

```
权益乘数 = 总资产 / 净资产
```

**含义**：用多少倍杠杆放大收益

**注意**：杠杆是把双刃剑，盈利时被放大，亏损时也被放大

---

## 四种 ROE 类型

| 类型 | 净利率 | 周转率 | 杠杆 | 评价 |
|------|--------|--------|------|------|
| **优质型** | 高 | 高 | 低 | 完美，护城河强 |
| **稳定型** | 高 | 中 | 中 | 稳健，价值型 |
| **周期型** | 波动大 | 波动大 | 中 | 随行业波动 |
| **危险型** | 低 | 低 | 高 | 高杠杆堆积，脆弱 |

## 实战应用

```
某公司 ROE = 30%，分解后发现：
- 净利率 5%（低）
- 周转率 1.5（低）
- 权益乘数 4（高杠杆）

结论：ROE 高是靠借债堆出来的，风险大
```
