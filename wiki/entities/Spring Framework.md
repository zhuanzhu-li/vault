# Spring Framework

[Spring Framework](Spring%20Framework.md) 是一个用于 Java 平台的广泛应用框架和控制反转容器。它为基于 Java 的现代企业应用程序提供了一个全面的编程和配置模型。

其生态系统最近新增了 [Spring AI](Spring%20AI.md)，它将 AI 工程模式直接引入到该框架中。该项目由 [Broadcom](Broadcom.md) 维护。
