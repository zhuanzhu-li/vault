# Apache 软件基金会

Apache 软件基金会（Apache Software Foundation，ASF）是一个成立于 1999 年的非营利性开源软件基金会，为超过 350 个开源项目提供组织、法律和财务支持。

## 托管项目

ASF 旗下包括众多知名开源项目，其中涉及大数据和分布式计算的关键项目包括：

- **[Apache HBase](Apache%20HBase_Apache%20HBase.md)** — 运行于 HDFS 上的分布式 NoSQL 数据库
- Apache Hadoop — 分布式存储与计算框架
- Apache Spark — 统一分析引擎
- Apache Kafka — 分布式消息流平台
- Apache Cassandra — 分布式 NoSQL 数据库

## 项目治理

- 每个 ASF 项目由独立的项目委员会（PMC）管理
- 采用精英治理（Meritocracy）模式——贡献者通过持续贡献获得提交者和 PMC 成员身份
- 使用 JIRA 跟踪问题，通过邮件列表进行项目沟通
