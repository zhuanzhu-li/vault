# GeoTools

GeoTools 是一个由 [[OSGeo_开源地理空间基金会]] 托管的开源 Java 库，提供了丰富的符合 [[OGC_开放地理空间信息联盟]] 规范的地理空间数据处理方法。

它是 Java 生态中构建 [[WebGIS]] 的主流底层工具库（著名的开源地图服务器 GeoServer 即构建于 GeoTools 之上）。其整体设计详见 [[GeoTools Architecture_GeoTools架构]]。
