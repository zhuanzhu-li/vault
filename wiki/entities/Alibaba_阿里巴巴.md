# Alibaba（阿里巴巴）

阿里巴巴集团是中国最大的电子商务公司之一，同时也是重要的开源技术贡献者。阿里巴巴技术团队通过《阿里巴巴Java开发手册》等规范，将内部技术经验回馈给广大开发者社区。

## 核心产品与技术

| 产品/技术 | 说明 |
|---------|------|
| **Alibaba Druid** | 数据库连接池，Java世界最流行的连接池实现之一 |
| **Alibaba FastJSON** | 高性能 JSON 序列化/反序列化库 |
| **Alibaba Nacos** | 服务发现、配置管理平台 |
| **Alibaba Sentinel** | 流量控制、熔断降级组件 |
| **Arthas** | Java 诊断利器，支持在线排查线上问题 |
| **p3c** | Java 代码规约检测插件，支持 IDE 和 CI 集成 |

## 阿里巴巴 Java 开发手册

《阿里巴巴Java开发手册》是阿里巴巴集团技术团队的集体智慧结晶，经历了多次大规模一线实战的检验。手册以Java开发者为中心视角，划分为六大维度：

1. **编程规约** — 命名风格、常量定义、代码格式、OOP规范
2. **异常日志** — 异常处理、日志规范
3. **单元测试** — 测试质量保障
4. **安全规约** — 鉴权、输入校验、敏感数据保护
5. **工程结构** — 应用分层、二方库依赖、服务器配置
6. **MySQL数据库** — 建表、索引、SQL、ORM规范

### 手册愿景

> **码出高效，码出质量**

现代软件架构需要协同开发完成，高效协作即降低协同成本、提升沟通效率。适当的规范和标准绝不是消灭代码内容的创造性、优雅性，而是限制过度个性化，以一种普遍认可的方式一起做事。
