# OGC (Open Geospatial Consortium)

开放地理空间信息联盟，是一个制定空间信息和基于位置服务（LBS）相关标准的国际组织。

在 [[WebGIS]] 领域，OGC 制定了多种常用的服务标准，如：
- WMS (Web Map Service)
- WFS (Web Feature Service)
- WMTS (Web Map Tile Service)
